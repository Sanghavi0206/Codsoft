# Quizly

Quizly is a responsive online quiz application built with HTML, CSS, and JavaScript. It allows users to answer multiple-choice questions, track progress, and view detailed results with category-wise performance analysis.

## Features

- Interactive multiple-choice quizzes
- Progress tracking
- Score calculation and analytics
- Category-based questions
- Answer review section
- Responsive design for all devices

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```text
Quizly/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/quizly.git
   ```

2. Open `index.html` in your browser or run it using a local server.
